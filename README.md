#### 3. Brightness Up
Increase the Brightness. 
```
AugImage = DocumentAugmention.BrightnessUp(BGRImage)
```
You can also specify the amount of Brightness to increase with
```
AugImage = DocumentAugmention.BrightnessUp(BGRImage,alpha=50.0)
```
